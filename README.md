# Tag Repair plugin for Melody and Movable Type v4.3x #

A plugin for MT 4.3x and Melody installations which fixes a particular kind of
corruption in `mt_tag` and `mt_objecttag` tables. 

You very likely don't need to use this plugin and should only attempt to if
you know **exactly** what you're doing.

## PLUGIN REQUIREMENTS ##

   * Any version of [Melody][] or [Movable Type v4.3x][MT]

[MT]:                   http://movabletype.org/
[Melody]:               http://openmelody.org/

## INSTALLATION ##

Unzip the download archive. Move the resulting folder to `$MT_HOME/plugins/`
(where `$MT_HOME` is your MT or Melody application directory).

If you use Git, you can do the following:

    cd $MT_HOME/plugins
    git clone git@github.com:endevver/mt-plugin-tag-repair.git

## CONFIGURATION ##

There is no configuration for this plugin.
